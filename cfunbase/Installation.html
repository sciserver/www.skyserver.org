CfunBASE Installation

Download CfunBASE.zip for accesing the project with the C# source code.
The API can be accesed here.

For uploading the CLR functions into an MS-SQL Server database, the uploading scripts can be found in the following subdirectories, according to the version of SQL server:
MS-SQL Server 2005: \SQL\UploadingScripts\SQL2005\
MS-SQL Server 2008: \SQL\UploadingScripts\SQL2008\
